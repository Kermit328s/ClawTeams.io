## 1. 板块职责

定义整个系统的数据地基，包括最小状态单元、数据库 Schema、存储基础设施和容器化标准。其他所有板块都依赖本板块的数据模型和基础服务。

## 2. 最小状态单元定义（优先级最高）

### 核心问题

一个龙虾完成一次任务之后，它必须留下什么，才能让下一个龙虾（或它自己下次）接得上？

**这个问题的答案决定整个系统的地基。必须在其他模块开发前先回答清楚。**

### 状态单元需包含

```json
{
  "task_id": "任务唯一标识",
  "agent_id": "执行龙虾",
  "state": "completed | failed | blocked | human_required",
  "result": "结构化输出",
  "artifact_ids": ["产出档案ID列表"],
  "cognitive_signal": "是否触发认知层判断",
  "context_snapshot": "执行时的上下文快照引用",
  "timestamp": "完成时间",
  "upstream_task_ids": ["上游依赖任务"],
  "downstream_task_ids": ["下游待触发任务"]
}
```

### 设计原则

- 强语义结构，不接受纯文本
- 历史不可篡改，只能追加
- 版本化存储，支持回滚（回滚 = 移动指针，不删除历史）
- 事件驱动，状态变更自动触发事件

## 3. 数据库设计

### Neo4j 图数据库（意图图谱 + 关系）

用于存储：
- 意图图谱（Intent Graph）：目标、任务、决策、人工节点及其关系
- 状态时间线（State Timeline）：actions log、progress states、diff log
- 知识沉淀（Knowledge Memory）：认知节点及演化关系
- 社交上下文（Social Context）：角色、负责范围、协作关系
- 龙虾账号关系：owner ↔ agent、team ↔ agent 绑定

### 节点类型

```
(:Goal)        — 目标节点
(:Task)        — 任务节点
(:Decision)    — 决策节点
(:Human)       — 人工节点
(:Cognition)   — 认知节点
(:Agent)       — 龙虾账号
(:User)        — 人类账号
(:Team)        — 团队
(:Role)        — 角色
```

### 边类型

```
[:DEPENDS_ON]      — 顺序依赖（A完成后才能做B）
[:PARALLEL_WITH]   — 并行关系
[:CONDITION]       — 条件依赖（满足条件X才触发）
[:AGGREGATES]      — 聚合关系（A和B都完成才触发C）
[:LOOP_BACK]       — 回环关系（不合格回到上游重做）
[:BELONGS_TO]      — 归属关系
[:OWNS]            — 所有权关系
[:RESPONSIBLE_FOR] — 负责关系
[:RELATES_TO]      — 认知关联
[:EVOLVED_FROM]    — 认知迭代
```

### 关系数据库（账号、会话、权限）

用于存储：
- 人类账号（JWT、会话管理）
- 龙虾账号（API Key、能力声明）
- 权限配置（自动推导结果 + 手动例外）
- 账号绑定关系

## 4. 对象存储

### Cloudflare R2（早期 / SaaS）

- 无出口流量费
- S3 兼容接口
- 用于存储：文档、报告、视频、大文件、数据集

### MinIO（私有化部署）

- 完全兼容 S3 接口，代码零改动
- 企业内网部署

### 存储分层

| 数据类型 | 存储方案 |
|---------|---------|
| 结构化数据、JSON | 图数据库直存 |
| 文档、报告 | R2 对象存储 + 元数据存图数据库 |
| 大文件、数据集、视频 | R2 + CDN + 预签名URL |
| Docker 镜像 | GitHub Container Registry / Harbor |

---

## 5. 容器化标准

### Docker 运行环境

- 龙虾运行在 Docker 容器中
- 跨机器迁移，一键恢复
- 镜像私有仓库（GitHub Container Registry）
- 版本化管理

### 本地部署规则

- 龙虾可在本地电脑运行
- 所有状态实时同步至团队大脑，本地不保留独占数据
- 本地仅作为执行容器（算力），不作为数据归宿
- 网络断开时允许本地临时缓存，重新联网后强制同步
